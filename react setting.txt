npm 으로 yarn 설치 (구글참고)

1. yarn add [옵션] 설치할패키지이름
2. yarn create react-app 프로젝트이름
3. SCSS다운 yarn add node-sass@4.14.1 open-color sass-loader classnames
4. styled-components 패키지 다운 yarn add styled-components
5. yarn 설치후 index.js에서 React.StrictMode 지우기
6. app.js를 class형으로 바꿔놓기 (보통 메인페이지는 class로 꾸려놓아야
함수 컴포넌트들에게 값을전달해주기위한 설정)

css초깃값 설정
*{
  list-style: none; /*목록태그들 없애기*/
  text-decoration: none; /*밑줄 지우기*/
  border-collapse: collapse;
  margin: 0;
  padding: 0;
  /* font-weight: 300; */
  color:black;
  letter-spacing: -1px;
}
.hide{
  position: absolute; left: -5000px; /*날려버리기*/
}
